새로운 노트북 구입을 위한 선택을 위한 하드웨어 사양 정리


노트북을 구입하려는 이유?
1. 현재 사용하는 노트북이 5년전 모델
2. 운영체제를 윈도우에 리눅스로 변경을 해도 하드웨어 성능 문제 현재 작업을 만족 시키지 못함

현재 진행 중인 공부 및 자기 개발
1. 딥러닝 - 영상 처리
2. opencv image processing
3. python data analysis
4. 임베디드 펌웨어 
5. 백엔드 또는 프론트엔드 선택에 따라 진행할 수 있다. 

노트북 하드웨어 사양
1. CPU - intel, amd, mac_CPU
2. RAM - 최소 16기가 이상. 원하는 랩 용량은 64기가
3. VGA - 딥러닝으로 내장 그래픽 카드 모델은 절대 안 삼. 외부 그래픽 카드를 지원하는 모델. 
4. Network - 무선 모델 선택. 유선 지원은 선택하지 않음. 얇은 노트북 사이즈를 선택
5. weight - 휴대용으로 사용할 거라 1.3Kg 미만이었으면 한다. 

맥북은 예외, 맥북은 CPU와 VGA는 intel과 amd 보다 좋다. 문제는 가격
